## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- git
- Broswer to Display
- IDE to run and edit the code

### Setup

- open a terminal
- run ``` git clone git@github.com:mr-abdellah/math-magicians.git```
- run ``` cd math-magicians```
- then run ``` npm install```
- then run ``` npm start```
